## 🚀 Projects

### [dux](https://github.com/nkroker/dux) — Webhook Forwarding Service
Self-hosted webhook proxy built in Ruby on Rails. Exposes global endpoints that forward to your local dev environment — like ngrok but for webhooks. JWT auth, Docker support, and a REST API for managing forwards.

### [tunnel](https://github.com/nkroker/tunnel) — Local Tunnel Tool
TypeScript tool for exposing a locally running server over the internet. Built as a self-hosted alternative to ngrok.

### [kuber-installer-ubuntu](https://github.com/nkroker/kuber-installer-ubuntu) — Kubernetes Setup Script
One-shot bash script to install and configure Docker + Kubernetes on Ubuntu 20.04+. Useful for spinning up dev clusters fast.
